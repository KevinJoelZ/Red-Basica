Red Básica

Qué es una red básica en Packet Tracer

Una red básica conecta dispositivos como routers, switches y equipos finales para permitir el intercambio de datos. Aprende los elementos principales y la lógica de diseño de esta topología.

Componentes de una red simple

En una red básica encontramos un router, un switch y dispositivos finales como una PC y una laptop. Estos componentes trabajan juntos para enrutar y conmutar paquetes dentro de una misma subred.

Conceptos clave:

  • Router: dispositivo que conecta redes diferentes y toma decisiones de encaminamiento.
  • Switch: conmutador de capa 2 que envía tramas a partir de direcciones MAC dentro de la LAN.
  • Host: equipo final, como una PC o laptop, que envía y recibe datos.
  • Subred: segmento de red definido por una máscara de subred y un rango de direcciones IP.

La red básica también permite estudiar cómo se asignan direcciones IP y cómo los paquetes se mueven entre dispositivos con ayuda del switch y del router.

En Cisco Packet Tracer, esta topología es útil para practicar conexiones físicas, configuración de interfaces y pruebas de conectividad.
