RIP versión 2

Enrutamiento dinámico con RIP v2

RIP v2 es un protocolo de enrutamiento dinámico que comparte rutas entre routers en redes pequeñas y medianas.

Ver contenido

¿Qué es RIP v2?

RIP versión 2 es un protocolo de vector de distancia que usa la métrica de saltos para determinar la mejor ruta. Envía actualizaciones periódicas de su tabla de enrutamiento a los vecinos mediante broadcasts o multicasts.

Características principales

  • Utiliza el conteo de saltos (hop count) como métrica.
  • Soporta máscaras de subred de longitud variable (VLSM).
  • Envía actualizaciones cada 30 segundos.
  • Soporta autenticación simple en versiones extendidas.

Configuración básica en Cisco IOS

enable
configure terminal
router rip
version 2
network 192.168.1.0
network 192.168.2.0
no auto-summary
end

Con estos comandos, el router comenzará a anunciar las redes configuradas a sus vecinos usando RIP v2.

Verificación de RIP

Para comprobar que RIP está funcionando correctamente, usa los siguientes comandos:

show ip protocols
show ip route
show ip rip database

Estas órdenes muestran el estado del protocolo, la tabla de rutas aprendidas por RIP y la base de datos de rutas.
