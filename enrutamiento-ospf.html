OSPF

Protocolo de enrutamiento de estado de enlace

OSPF es un protocolo dinámico de enrutamiento recomendado para redes grandes, con áreas y convergencia rápida.

¿Qué es OSPF?

OSPF (Open Shortest Path First) es un protocolo de enrutamiento de estado de enlace que crea un mapa completo de la topología de la red y calcula la ruta más corta utilizando el algoritmo SPF.

Conceptos clave

  • Área: división lógica de la red para escalar OSPF y reducir la cantidad de información de enrutamiento.
  • Router vecino: routers que intercambian información OSPF en una misma red.
  • Base de datos LSDB: almacena información de los enlaces y topología OSPF.
  • Algoritmo SPF: calcula la ruta más corta a partir de la información de la LSDB.

Configuración básica en Cisco IOS

enable
configure terminal
router ospf 1
network 192.168.1.0 0.0.0.255 area 0
network 192.168.2.0 0.0.0.255 area 0
end

Con estos comandos, el router anunciará las redes dentro del área 0 y establecerá relaciones de vecino con otros routers OSPF.

Verificación de OSPF

Utiliza estos comandos para revisar el estado de OSPF y las rutas aprendidas:

show ip ospf neighbor
show ip route ospf
show ip ospf database

Estos comandos ayudan a validar que OSPF ha formado adyacencias y que las rutas internas se han propagado correctamente.
