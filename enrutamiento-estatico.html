Enrutamiento Estático

Rutas manuales para conectar redes

El enrutamiento estático permite configurar rutas fijas en los routers para definir cómo se envía el tráfico entre segmentos de red.

Ver contenido

¿Qué es el enrutamiento estático?

El enrutamiento estático es un método en el que el administrador configura manualmente rutas en el router. Estas rutas no cambian automáticamente y se utilizan cuando la topología de red es simple o cuando se desea control total sobre el camino de los paquetes.

Ventajas y desventajas

  • Ventajas: configuraciones fáciles, menos recursos de CPU en el router y mayor control sobre las rutas.
  • Desventajas: no se adapta automáticamente a cambios de red y requiere actualización manual en cada router.

Ejemplo de configuración

Imagina dos routers conectados y dos redes detrás de ellos. La ruta estática permite que cada router sepa cómo alcanzar la red remota.

configure terminal
ip route 192.168.2.0 255.255.255.0 10.0.0.2
end

Donde:

  • 192.168.2.0 255.255.255.0 es la red destino.
  • 10.0.0.2 es la dirección IP del siguiente salto o vecino.

Verificación

Después de configurar una ruta estática, comprueba su funcionamiento con:

show ip route
ping 192.168.2.1

El comando show ip route mostrará la ruta estática en la tabla de enrutamiento, y ping ayudará a validar la conectividad.
